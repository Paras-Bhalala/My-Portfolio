# My-Portfolio
# 🚀 Paras Bhalala | Portfolio Website

A modern, responsive portfolio website built to showcase my projects, skills, and professional journey.

**Live Demo:** [👉 View My Portfolio](https://parasbhalala.vercel.app/)

---

## 🛠️ Built With
- **HTML5** - Semantic structure
- **Tailwind CSS** - Modern styling and responsiveness
- **GitHub Pages** - Hosting
- **Lucide Icons** - Clean iconography

## ✨ Key Features
- **Responsive Design:** Optimized for mobile, tablet, and desktop screens.
- **Project Showcase:** A clean grid layout for my best work.
- **Technology Proficiency:** A visual representation of my current tech stack.
- **Education Timeline:** Highlights of my academic background.
- **Interactive Navigation:** Smooth scrolling and easy access to my CV.
